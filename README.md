## Description
This repository serves as an electronic logbook containing all C programming scripts, class exercises, and foundational logic written during lecture sessions. It acts as a structured digital portfolio for end-of-semester grading.

## Topics Covered
* **Basic Strings:** Utilizing `strcpy` and string arrays.
* **Standard Input/Output:** Reading data using `scanf`, `printf`, and safe string handling with `fgets`.
* **Preprocessor Directives:** Defining global constants and macros (`#define`).
* **Conditionals & Logic:** Structuring `if`, `else if`, nested statements, and logical AND (`&&`) operators.
* **Advanced Conditionals:** Implementing ternary operators (`? :`) and `switch` statements for specific case evaluations.
* **Iteration:** Building standard `while`, guaranteed `do-while`, and both incrementing/decrementing `for` loops.
* **Modular Programming:** Declaring function prototypes and writing custom user-defined functions.
